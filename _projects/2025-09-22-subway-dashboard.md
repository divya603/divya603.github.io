A real-time big data processing pipeline for MTA subway data. The system ingests GTFS and JSON feeds, streams via Kafka, processes with Spark Structured Streaming, stores in MongoDB, serves through a FastAPI backend, and visualizes on a React dashboard.

### Highlights
- Ingestion from MTA GTFS and JSON feeds
- Real-time processing with Kafka + Spark
- Persistent storage in MongoDB
- API layer using FastAPI
- Interactive React dashboard with filtering and analytics
